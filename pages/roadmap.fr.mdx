# Feuille de route

La feuille de route concerne l'amélioration globale de la documentation, tout en gardant une base réduite, afin qu'elle soit facile à maintenir et que nous ne perdions pas l'objectif de ce site : être léger.

### Nouvelles sections à venir :

- Concepts de base de Drupal pour les développeurs : liste des concepts les plus utilisés et documentation sur la création d'un module minimal et d'un thème
- Prérequis Drupal pour les développeurs : ajouter de la documentation sur les ressources pour apprendre PHP, Symfony, Twig, Composer, Git, ...
- Gestion de projet Drupal et budget
- Déploiement en production et bonnes pratiques d'environnement (robots.txt / headers, indexation, staging, CDN, CI/CD, ...)
- Une courte liste de cas pour le commerce, le découplage, le multilingue, ... afin qu'il soit facile de trouver un bon exemple pour une première comparaison avec d'autres solutions

Les mises à jour sont publiées sur [Twitter](https://twitter.com/drupalship) et [Mastodon](https://drupal.community/@drupalship)
