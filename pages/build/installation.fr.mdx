# Installation du noyau Drupal et des projets contribués

Installez et exécutez Drupal localement en un rien de temps.

## Installation locale rapide

C'est parfait pour évaluer et développer localement avec Drupal.
Une installation qui ne prend pas plus de 5 minutes. Vous devriez avoir PHP 8.1 et Composer déjà installés.

### Requérir le projet Drupal

```bash
composer create-project drupal/recommended-project my-site
```

### Installer Drupal

```bash
cd my-site
```

Il y a 2 scripts : `install` et `quick-start`.

`quick-start` est une enveloppe autour de `install` avec quelques raccourcis par défaut comme l'exécution d'un serveur PHP local, l'ouverture du navigateur et la connexion en tant qu'administrateur.
Nous montrerons ici les deux options, car il est bon de savoir ce qui se passe sous le capot.

#### Installation standard

Installation avec une base de données SQLite et le profil standard.
C'est parfait pour tester une installation Drupal vierge et installer quelques modules pour évaluation.
De plus, cette configuration est le candidat idéal pour déboguer un problème de projet Drupal en isolation et contribuer.

```bash
php web/core/scripts/drupal install standard
```

#### Installation avec du contenu de démonstration traduit

Installation avec le profil Demo Umami.
Il est livré avec du contenu de démonstration, des traductions et quelques autres modules activés. Utilisez cela si vous voulez évaluer un ensemble de fonctionnalités Drupal plus complet.

```bash
php web/core/scripts/drupal quick-start demo_umami
```

### Exécutez-le avec le serveur intégré PHP

Vous pouvez ignorer cela si vous avez utilisé le script `quick-start`.

```bash
cd web
php -S 127.0.0.1:8888
```

### Installez des projets contributifs

```bash
composer require drupal/[nom_du_projet]
```
Puis activez le projet avec [Drush](/develop/toolkit-for-drupal-developers#drush-cli) ou l'UI.

#### Activer un projet avec l'UI

- Modules /admin/modules
- Thèmes /admin/appearance

#### Activer un projet avec Drush

```bash
drush en [nom_du_projet]
```
